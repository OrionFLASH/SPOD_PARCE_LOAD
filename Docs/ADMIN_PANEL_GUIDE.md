# Админ-панель SPOD: краткий гид

Документ объединяет краткие материалы по админ-панели: назначение, запуск, статус и ограничения.

## Назначение

Админ-панель предназначена для просмотра и редактирования данных SPOD через веб-интерфейс с проверками связей, валидацией и обработкой JSON-полей.

## Базовый запуск (если используется отдельный проект `admin_panel`)

```bash
cd admin_panel
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

Открыть в браузере: `http://localhost:5000`.

## Основной функционал (исторически реализованный)

- вкладки по файлам;
- поиск, сортировка, пагинация;
- CRUD-операции;
- проверки ключей и связей;
- базовое редактирование JSON.

## Важные ограничения

- Материалы по админ-панели в `Docs/` частично исторические и могут расходиться с текущей реализацией.
- Актуальным источником поведения является код самого проекта админ-панели.
- Для рабочих решений по текущему пайплайну SPOD ориентироваться на `README.md` и документы `CONSISTENCY_*`.
