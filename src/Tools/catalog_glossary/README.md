# Фрагменты глоссария для `SPOD_INPUT_DATA_CATALOG.md`

Файлы в этой папке подключаются скриптом `src/Tools/build_spod_input_catalog.py` к разделам **REWARD** и **CONTEST** после машинного разбора JSON.

- `REWARD_ADD_DATA_glossary.md` — матрица полей, дерево, алфавитный справочник, `getCondition`.
- `CONTEST_FEATURE_glossary.md` — то же для `CONTEST_FEATURE`.

При изменении структуры JSON в данных правьте глоссарии здесь, затем пересоберите каталог.
