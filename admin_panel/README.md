# SPOD Admin Panel

Административная панель для редактирования данных SPOD.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Убедитесь, что файлы данных находятся в каталоге `SPOD/`

## Запуск

```bash
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000

## Структура

- `app.py` - главное приложение Flask
- `config.py` - конфигурация файлов и настроек
- `utils/` - утилиты для работы с данными
  - `file_manager.py` - управление файлами
  - `data_manager.py` - управление записями
  - `json_editor.py` - редактирование JSON полей
- `templates/` - HTML шаблоны
- `static/` - статические файлы (CSS, JS)

## Функциональность

- Просмотр данных из всех CSV файлов
- Добавление, редактирование, удаление записей
- Редактирование JSON полей
- Поиск и сортировка
- Пагинация
- Валидация данных
- Каскадное удаление зависимых записей

## Сессии редактирования

Все изменения сохраняются в каталоге `EDIT/YYYYMMDD_HHMM/`. 
Исходные файлы в `SPOD/` не изменяются.

Резервные копии создаются в `BACKUP/` перед каждым изменением.
