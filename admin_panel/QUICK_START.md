# Быстрый старт

## Установка и запуск

1. Перейдите в каталог admin_panel:
```bash
cd admin_panel
```

2. Запустите скрипт:
```bash
./start.sh
```

Или вручную:
```bash
# Создайте виртуальное окружение (если еще не создано)
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate

# Установите зависимости
pip install -r requirements.txt

# Запустите приложение
python app.py
```

3. Откройте в браузере: http://localhost:5000

## Структура данных

- Исходные файлы: `../SPOD/` (не изменяются)
- Сессии редактирования: `../EDIT/YYYYMMDD_HHMM/`
- Резервные копии: `../BACKUP/`

## Функции

- ✅ Просмотр всех файлов через вкладки
- ✅ Поиск и сортировка
- ✅ Добавление записей
- ✅ Редактирование записей
- ✅ Удаление записей (с проверкой зависимостей)
- ✅ Редактирование JSON полей
- ✅ Валидация данных
- ✅ Пагинация

## Примечания

- Все изменения сохраняются только в каталоге EDIT
- Исходные файлы в SPOD не изменяются
- Перед каждым изменением создается резервная копия
